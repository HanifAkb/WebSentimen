Cara Menggunakan

1

Membuat Analisis

  1. Buka menu Buat Analisis.
  2. Pilih salah satu tab:
    Pengumpulan Data X: isi API Key, versi model, kueri, dan rentang tanggal.
    CSV/TXT: unggah file lalu pilih kolom teks yang ingin dianalisis.
  3. Klik Jalankan untuk memulai proses.
Jangan bagikan API key Anda ke orang lain.
2

Melihat Riwayat

  1. Buka menu Riwayat.
  2. Pilih tab Riwayat Pengumpulan Data X atau Riwayat CSV/TXT.
  3. Klik salah satu data untuk membuka detail hasil.
  4. Gunakan halaman detail untuk melihat tabel, dashboard, atau unduh file hasil jika tersedia.
  5. Jika proses pengumpulan data belum selesai, tunggu hingga status selesai atau pantau dari halaman detail riwayat.
