Beranda

Selamat Datang di Website Analisis Sentimen

Website ini digunakan untuk analisis sentimen menggunakan dua model sekaligus: KNN dan SVM. Anda bisa menganalisis satu kalimat, file CSV/TXT, atau data hasil crawling Twitter.

Website Ini Untuk Apa?

  • Membandingkan hasil klasifikasi KNN dan SVM pada data yang sama.
  • Memproses input manual (kalimat tunggal) dan input batch (CSV/TXT).
  • Scraping data X via twitterapi.io lalu langsung klasifikasi + unduh hasil CSV.

Cara Menggunakan

Berikut adalah cara penggunaan setiap menu yang ada pada website:

A
Menu "Buat Analisis"
  1. Buka menu Buat Analisis.
  2. Pilih input kalimat tunggal atau unggah CSV/TXT.
  3. Masukkan kalimat yang ingin diprediksi atau ungguh file CSV/TXT dan pilih kolom kalimat yang ingin diprediksi .
  4. Klik analisis untuk melihat hasil KNN dan SVM.
B
Menu "Scraping Web X"
  1. Buka twitterapi.io lalu login/daftar.
  2. Masuk ke Menu Dashboard, lalu lihat bagian User Information.
  3. Salin bagian API Key.
  4. Kembali ke menu Scraping Web X di website ini, lalu tempel API key.
  5. Isi kueri, bahasa (opsional), dan rentang tanggal.
  6. Klik mulai scraping untuk melihat hasil klasifikasi.
Jangan bagikan API key Anda ke orang lain.
C
Menu "Riwayat"
  1. Buka menu Riwayat.
  2. Pilih bagian Riwayat Scraping atau Riwayat Prediksi.
  3. Klik salah satu baris riwayat untuk melihat detail hasil.
  4. Pada detail scraping, lihat tabel hasil, tampilkan dashboard, atau unduh CSV jika tersedia.
  5. Jika status scraping belum selesai, klik Lanjutkan Scrap untuk melanjutkan proses.
  6. Centang riwayat lalu klik Hapus jika ingin menghapus data yang dipilih.
